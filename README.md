📊 Global Terrorism Analysis (1970–2017)



📌 Project Overview

This project analyzes global terrorism data from 1970 to 2017 to identify patterns, trends, and insights related to terrorist activities across different regions of the world.

The analysis helps in understanding how terrorism has evolved over time and highlights the most affected areas, attack types, and targets.




🎯 Objectives


   -To analyze trends in terrorist incidents over the years

   -To identify the most affected countries and regions

   -To study different types of attacks and targets

   -To visualize patterns for better understanding

   -To extract meaningful insights using data analysis techniques




🛠️ Technologies Used


   -Python

   -Pandas

   -NumPy

   -Matplotlib

   -Seaborn

   -Google Collab




📊 Data Analysis & Visualization


  The following analysis has been performed:


  -Year-wise trend of terrorist attacks

   -Country-wise analysis

   -Region-wise comparison

   -Attack types distribution

   -Casualty analysis

   -Top affected countries

   -Graphs and visualizations are used to represent data clearly and effectively.




📈 Key Insights


  -Terrorist activities have increased significantly after the year 2000

  -Certain regions like the Middle East and South Asia are highly affected

  -Bombings and armed assaults are the most common attack types

  -Specific countries experience repeated incidents over time
